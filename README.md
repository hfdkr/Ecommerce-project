#  Ecommerce Cart Page

A modern and responsive Ecommerce Cart Page built using HTML and Tailwind CSS.

##  Overview

This project represents a shopping cart interface for an e-commerce website. It includes:

- Promotional banner
- Navigation bar
- Product search input
- Shopping cart items
- Quantity controls
- Order summary section
- Checkout button
- Footer with company information and social links
