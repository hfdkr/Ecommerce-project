#  Ecommerce Cart Page

A modern and responsive Ecommerce Cart Page built using HTML and Tailwind CSS.

##  Overview

This project represents a shopping cart interface for an e-commerce website. It includes:

- Promotional banner
- Navigation bar
- Product search input
- Shopping cart items
- Quantity controls
- Order summary section
- Checkout button
- Footer with company information and social links


##  Features

✅ Responsive Layout

✅ Modern UI Design

✅ Shopping Cart Interface

✅ Product Quantity Controls

✅ Order Summary Calculation Section

✅ Tailwind CSS Styling

✅ Clean and Structured Code

##  Technologies Used

- HTML5
- Tailwind CSS
- Google Fonts
- CSS Custom Properties
